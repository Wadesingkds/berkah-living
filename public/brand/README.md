## Logo Files

| File | Description | Usage |
|------|-------------|-------|
| `logo-icon.svg` | Icon only version | Favicon, app icon, social media |
| `logo-icon-dark.svg` | Icon for dark backgrounds | Dark UI themes |
| `logo-vertical.svg` | Logo with vertical text | Print materials, headers |
| `logo-horizontal.svg` | Logo with horizontal text | Website headers, banners |
| `logo-white.svg` | White logo version | Dark backgrounds |

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|

| Golden Amber | `#D4A041` | Secondary, highlights |
| Forest Green | `#5C7C58` | Success, nature theme |
| Warm Cream | `#F4E4BC` | Backgrounds, light accents |

## Usage Guidelines

1. Always use the latest version of brand assets
2. Maintain aspect ratio when resizing
3. Keep clear space around logos
4. Use brand colors as specified
5. Follow typography scale for consistency
